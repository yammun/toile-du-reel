Exposition

1. Action de mettre en vue ; fait d'être montré, exposé : Exposition d'un mort sur un lit de parade. Synonyme : présentation

2. Étalage pour la vente : Exposition de livres dans la vitrine.

3. Présentation au public d'œuvres d'art ; ensemble des œuvres exposées : Exposition de peinture.

4. Présentation de produits divers de l'industrie, de l'agriculture, etc. : Exposition de machines-outils. L'exposition universelle. Synonymes : concours - foire - salon

5. Endroit où se font ces présentations : Entrer à l'exposition.

6. Mise en vente en grande quantité d'un ensemble d'articles de même type dans les grands magasins : Exposition de blanc.

7. Orientation d'un édifice, d'une pièce, d'un tableau, etc., considérée par rapport à la manière dont ils reçoivent la lumière, le jour, le soleil : L'appartement a une double exposition. Synonyme : situation

8. État de quelqu'un, de quelque chose qui est soumis à l'influence, à l'action de quelque chose : L'exposition au froid.

9. Action d'expliquer, de développer, de faire connaître ; exposé : Faire l'exposition de sa doctrine. Synonymes : explication - exposé

Littérature10. Début d'une œuvre littéraire, et en particulier d'une œuvre dramatique, dans laquelle on expose le sujet. Synonymes : introduction - protase

Liturgie11. Cérémonie qui consiste à montrer aux fidèles ce qui est offert à leur vénération.

Musique12. Dans une composition, section dans laquelle le thème ou les différents éléments thématiques sont présentés.

Photographie13. Action d'exposer une surface sensible ; action d'un rayonnement sur une surface sensible.
